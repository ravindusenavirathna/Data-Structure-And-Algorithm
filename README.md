Data-Structure-And-Algorithm
============================

A comprehensive collection of C programs covering fundamental data structures and algorithms, along with key C programming basics. This repository is organized by topics, including arrays, linked lists, stacks, queues, recursion, control statements, functions, and more.

Repository Structure
--------------------

*   **Arrray**: Programs for array operations like insertion, deletion, and sorting.
    
*   **C Basics**: Step-by-step examples for beginners to learn C programming fundamentals, including data types, operators, control statements, loops, and functions.
    
*   **Linked List**: Various types of linked lists (singly, doubly, circular) with insertion and traversal implementations.
    
*   **Queue**: Array and linked-list implementations of queues, including circular queues.
    
*   **Recursion**: Classic recursion examples like counting digits, Fibonacci series, array printing, and string reversal.
    
*   **Stack**: Array-based and linked-list-based stack implementations.
    

Each folder contains C source files (`.c`) with corresponding executable files (`.exe`). The repository is ideal for beginners aiming to strengthen their understanding of data structures, algorithms, and basic C programming concepts through practical, hands-on examples.

Getting Started
---------------

1.  **Clone the repository**:
```bash
git clone https://github.com/ravindusenavirathna/Data-Structure-And-Algorithm
```
    
2.  **Compile and run** each C file using a C compiler to explore various data structures and algorithms in action.
    

Requirements
------------

*   C Compiler (e.g., GCC)
